# 🚀 SHIHAB UNIVERSE — Portfolio

A modern, interactive developer portfolio built with **Next.js**, featuring smooth animations, immersive visuals, and dynamic project showcases.

---

## ✨ Features

- ⚡ Built with Next.js (App Router)
- 🎬 Smooth animations using Framer Motion
- 🎥 Video-integrated hero section
- 🧩 Modular component-based architecture
- 📱 Fully responsive design
- 🌌 Interactive UI (Three.js scene integration)
